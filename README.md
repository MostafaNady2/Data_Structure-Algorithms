## **Setup Instructions**
### **For C++:**
1. Compile any file using g++:
   ```sh
   g++ -o output_file source_file.cpp
   ```
2. Run the compiled program:
   ```sh
   ./output_file
   ```

### **For Java:**
1. Compile the Java file:
   ```sh
   javac FileName.java
   ```
2. Run the program:
   ```sh
   java FileName
   ```
